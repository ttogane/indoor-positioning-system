# indoor-positioning-system
wifiルーターの電波強度を利用した室内位置情報システム

# 利用方法
### 以下環境が必要
OS: mac  
python : version 3.8.X  
wifiアクセスポイント: 3個以上  
室内の間取り図: `src/image/floor_plan.png`
